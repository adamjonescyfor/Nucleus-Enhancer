Load Template Files

Pick a folder of .txt templates to use in Nucleus.

Each .txt file becomes a template named after the file. This replaces your current local templates on this device. Official Salesforce templates always take priority and can't be overridden.

Loaded

    This loader runs in a normal tab so folder selection works reliably on every operating system (the extension popup can close when the system folder dialog opens, which is why loading from the popup may not work on Linux).
