            Help & tips
            ▾
        
        
      
            
    • Insert a template — click the 📄 button or right-click inside a Notes / Forensic Strategy box. Alt+T opens the menu, Alt+Z undoes an insert.
      • 
            
    • Quick-fill — right-click any date or time field for now; Completed By / Sealed By / Conducted By for you; Exhibit Type auto-picks from the exhibit name; and a Forensic Case lookup grabs the latest case.
      • 
            
    • Record navigation — Alt+← / → moves between process records in list order.
      • 
            
    • Columns — drag the column names above to reorder list-view tables; save layouts as presets.
      • 
            
    • Photographs — a "Download All Photographs" button appears on Uploaded Documents sections.
      • 
            
    • Notes — notes columns in list views are expanded and formatted automatically (toggle above).
      • 
            
    • Templates — your team's official templates sync automatically (about every 20 minutes); "Sync Now" forces it. "View Templates" shows versions and history. Admins get a full rich-text editor and can assign a template to several teams.
      • 
            
    • Read & understood — when a template is a controlled document, open it in View Templates and click "I have read & understood vX" to sign off the current version (a new version asks again). Admins get an Acknowledgements tab showing who is still outstanding.
      • 
            
    • Suggest an edit — in View Templates, click Suggest edit to propose a change with a reason. A template admin reviews it (with a diff) and applies or rejects it — nothing changes until they do.
      • 
            
    • Case report — export a sanitised disclosure report from any Forensic Case page.
      • 
            
    • Something not working? Open Diagnostics above, turn on Capture diagnostic log, reproduce the problem, then Download log and send the file to Adam — it helps pin down the cause fast.
